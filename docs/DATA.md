# Data

Sample volatility surfaces used in this project are derived from publicly
available option market data.

- **CBOE Volatility Surface Data**  
  https://datashop.cboe.com/volatility-surfaces
  Used for: `data/VolSurface_SPY_04072011.csv`

All data is used for research and illustrative purposes only. New data files
must come from public sources and should document their provenance here.
